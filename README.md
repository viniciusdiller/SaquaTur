# SaquaTur Frontend (Estático)

Projeto frontend estático da experiência turística de Saquarema, com foco em identidade visual, responsividade e navegação institucional.

## Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS

## Rodando localmente

```bash
npm install
npm run dev
```

## Build estático

```bash
npm run build
```

A aplicação gera saída estática (sem backend e sem integrações externas).
